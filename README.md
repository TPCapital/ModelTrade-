# Sea Trading OS v5.1

> 把交易决策压缩成四个动作：看什么 / 等什么 / 做什么 / 不做什么

基于真实交易复盘提炼的多品种训练终端，集成风控脊柱、QQQ期权专项系统、黄金现货、EUR/USD外汇和正股低频配置。

---

## 快速部署

```bash
npm install
npm run dev        # 本地预览 → http://localhost:5173
npm run build      # 生产构建 → dist/
```

部署到 Vercel / GitHub Pages：将 `dist/` 目录推送或直接连接仓库。

## 核心参数（$1,000账户）

| 参数 | 数值 | 说明 |
|------|------|------|
| 每笔止损 | $20 | 账户2%，无条件市价出场 |
| 每笔目标 | $40 | 1:2 R:R |
| 日亏熔断 | $50 | 账户5%，当日停止 |
| 时间止盈 | 45分钟 | 持仓超时无论盈亏出场 |
| 合约张数 | 固定1张 | 账户重建阶段不加张 |
| 主力标的 | QQQ | 暂不碰个股 |

## 四项入场条件（缺一不可）

1. **VWAP方向** — 价格明确站上/下VWAP，不横跳
2. **EMA趋势** — 9EMA在21EMA上/下方且发散
3. **成交量** — 当前量 > 前5根K线平均量 × 1.5倍
4. **时间窗口** — 09:45–11:30 ET

---

## 账户重建阶段目标

现阶段目标不是盈利，是建立稳定执行的肌肉记忆。

**评判标准**：这笔交易进场前，是否走完了全部检查清单？

连续稳定执行 **20笔** 后，再讨论张数和标的扩展。
