# Specularis Trading Model v5.8.1

Deployment-ready React + Vite training app for QQQ options, gold, EUR/USD, risk sizing, GEX setup, macro filters, and discipline drills.

## Local Development

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Vercel

Use the included `vercel.json`:

- Framework: Vite
- Install Command: `npm install`
- Build Command: `npm run build`
- Output Directory: `dist`

The app is a browser-only SPA. GEX setup values are stored in localStorage and are not uploaded.
